Bildformat inom HTML, SCSS och JavaScript: HTML:
Inom HTML kan du använda img-elementet för att bädda in bilder. Vanliga bildformat som stöds inkluderar JPEG, PNG, GIF och SVG. SCSS (Sass): SCSS är en preprocessor till CSS och påverkar inte de bildformat du kan använda. Det stöder CSS-regler för att stilisera och positionera bilder. JavaScript: Inom JavaScript kan du använda bildobjekt, canvas-elementet och olika API:er för att manipulera och arbeta med bilder. WebP är ett modernare bildformat som stöds av vissa webbläsare och kan användas för att minska filstorlek med bibehållen kvalitet.
